Note-Taker

What is note-taker?

Note-taker is a service that allows you to create course notes conveniently through your phone and/or desktop that are converted into pdfs hosted on Google Drive. We cater specifically to those in the academic field looking to create nice, simple course notes.

Sounds good! How do I get started?

Reading this page is the first step! Just proceed to the rest of this guide!

<ol>

An ordered list provides a numbered list. This tag must contain <li> elements. For instance, <ol> <li>First element </li> </ol>

<ul>

An unordered list provides a bullet list. This tag must contain <li> elements. For instance, <ul> <li>First element </li> </ul>

<dl>

An ordered list provides a formatted definition list. This tag must contain <li> elements. For instance, <dl> <li>First element </li> </dl>

<li>

This tag is defines a list element. Tags in which list elements are necessary include: <toc>, <ol>, <ul>, <dl>

<table>

Create a new table with this tag. In order for elements to appear, use <tr> and <td> table element tags

<tr>

Defines a new row in the table. For multiple column tables, include the <td> table element tag which defines new columsn within the table row.

<td>

Defines a new column in the table. This table element can be included in <tr> to create multiple column rows.
